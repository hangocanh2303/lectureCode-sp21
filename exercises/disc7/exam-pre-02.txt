2 Flip Flop
Suppose we have the flip function as defined below. Assume the method unknown
returns a random integer between 1 and N, exclusive, and runs in constant time.
For each definition of the flop method below, give the best and worst case runtime
of flip in Θ(.) notation as a function of N.
public static void flip(int N) {
    if (N <= 100) {
        return;
    }
    int stop = unknown(N);
    for (int i = 1; i < N; i++) {
        if (i == stop) {
            flop(i, N);
            return;
         }
     }
 }
(a)
public static void flop(int i, int N) {
    flip(N - i);
}
Best Case: Θ(N), Worst Case: Θ(N)

(b)
public static void flop(int i, int N) {
    int minimum = Math.min(i, N - i);
    flip(minimum);
    flip(minimum);
}
Best Case: Θ(1), Worst Case: Θ(NlogN)

(c)
public static void flop(int i, int N) {
    flip(i);
    flip(N - i);
}
Best Case: Θ(N), Worst Case: Θ()
